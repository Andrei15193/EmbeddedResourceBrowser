TryGetValue Method

Gets the item having the specified name.

This is a public method.

Parameters

  • name: string

    The name of the item to retrieve.

  • value: T

    The retrieved item, if found; otherwise default(T) is returned.

Returns: bool

If an item with the provided name exists, returns true; Otherwise false.

Exceptions

  • ArgumentNullException

    Thrown when name is null
